# UK-Fuel-Poverty-Dashboard-PowerBI
Power BI dashboard analyzing UK fuel poverty trends (2010-2024) with strategic insights for social impact programs. Features multi-table data modeling, DAX time intelligence measures, and regional deployment analysis for vulnerable household support services.

## Overview
Interactive Power BI dashboard analyzing 15 years of UK fuel poverty statistics (2010-2024) to inform strategic deployment of social support services.

## Features
- Multi-table star schema data model
- 5+ DAX measures with time intelligence (YoY analysis)
- Regional priority analysis
- Vulnerable household segmentation
- Brand-aligned visualizations

## Data Source
UK Government Fuel Poverty Statistics - Department for Energy Security & Net Zero (Open Government Licence v3.0)

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor

## Key Insights
- 43% reduction in fuel poverty since 2010
- West Midlands & London show highest aggregate gaps
- Private renters & single parents most vulnerable
